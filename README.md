## Getting Started

 These are the steps you need to follow to install all the dependencies and get the program running:


### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Baccount/project.git
   ```
2. Install dependencies
   ```sh
   pip install -r requirements.txt
    ```
